User Manual: How to Navigate

This project was completed 12/13/24 in Team Edge - SM

Objective:

The objective of this game is to collect six necessary ingredients and enter Sunset Hallow successfully, to fulfill the recipe. You do this by identifying the materials in the recipe book and using the address bar search query to navigate to different locations. The searches are written in the format: "/api/v1/search?enter=[location]" where the location does not include apostrophies.

Game's Contents:

The game is made up of a few pages:

    The Root: The starting point of this game, providing context for how you come across the recipe book.
    The Help Page (this one!): A user manual which teaches the user how to navigate in this game and hopefully answers any questions
    Recipe: Tells the user which locations to search for.
    Portal Page: This page is only accessible to show the search query results. It is the page which the user enters when searching up a location and progressing the story.

Locations

These are the locations must search up in the query and the items they are related to:

    Mom's House
    The Cozy Cabin
    The Easter Bunny
    The Wooded Peaks
    Windy Fields
    Dragon's Cove
    Sunset Hallow

Remember: when you search these up in the search query, do not include apostrophies!

Example: http://127.0.0.1:5000/api/v1/search?enter=moms%20home
